# Images Directory

Place your application images here.

## Required Images

1. **icon.png** (1024x1024 px) - Main app icon
2. **adaptive-icon.png** (1024x1024 px) - Adaptive icon for Android
3. **splash.png** (1284x2778 px) - Splash screen image
4. **favicon.png** (48x48 px) - Favicon for web
5. **notification-icon.png** (96x96 px) - Icon for push notifications

## Guidelines

- Use PNG format
- Ensure images are properly sized
- Keep file sizes reasonable (under 500KB for icons, under 2MB for splash)
- Use transparent backgrounds where appropriate

## Tools

You can use these tools to create icons:

- [Canva](https://www.canva.com/) - Easy online design
- [Figma](https://www.figma.com/) - Professional design tool
- [Expo Icon Generator](https://icon.kitchen/) - Specialized for React Native
- [GIMP](https://www.gimp.org/) - Free image editor

## Temporary Placeholders

For development, you can use placeholder images. The app will still run without them, but you'll see default Expo icons.
